# iris_fisher_api
установка:
откройте Powershell в каталоге, а затем скопируйте и вставьте все эти команды в терминал

    python -m venv .venv

    .venv\Scripts\Activate.Ps1

    python -m pip install --upgrade pip

    pip install -r requirements.txt

    uvicorn main:app 
использование

    .venv\Scripts\Activate.Ps1
    uvicorn main:app
